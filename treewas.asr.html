treewas.asr.fitch_downpass(genes, tree)[source]

The first pass of Fitch’s parsimony algorithm.

Returns preliminary ancestral state sets and per locus parsimony scores.

Parameters:

  • genes (pandas.DataFrame) – A DataFrame containing the observed binary genotypes. Each row represents a genetic locus and each column represents an individual from the sample

  • tree (TreeWrapper) – The reconstructed phylogeny. Names of leaf nodes need to correspond to the names of the individuals. Has to be bifurcating.

Returns:

A tuple containing a DataFrame with the ancestral state sets determined during the first pass. Ambiguous states are represented as the number 2. The second item in the tuple is a Series with the per locus parsimony scores.

Return type:

tuple[pandas.DataFrame, pandas.Series]

treewas.asr.fitch_parsimony(genes, tree, get_scores=False)[source]

Perform Fitch’s parsimony algorithm

Reconstructs ancestral state sets using Fitch’s algorithm [1] and if get_scores is True, the per locus parsimony scores will be returned.

For genes where the ancestral state is ambiguous, the entry is set to 0.5, otherwise the entry is either one or zero.

Note

Only works for binary states and bifurcating trees.

Parameters:

  • genes (pandas.DataFrame) – A DataFrame containing the observed binary genotypes. Entries either have to be boolean or only zeros and ones. Each row represents a genetic locus and each column represents an individual from the sample.

  • tree (TreeWrapper) – The reconstructed phylogeny. Names of leaf nodes need to correspond to the names of the individuals. Has to be bifurcating.

  • get_scores (bool) – If set to True, returns the per locus parsimony scores

Returns:

A DataFrame with reconstructed states and if get_scores is True a Series with the per-site parsimony score

Return type:

pandas.DataFrame | tuple[pandas.DataFrame, pandas.Series]

Raises:

NotImplementedError – if genes is not binary

References

[1]

W. M. Fitch, Toward Defining the Course of Evolution: Minimum Change for a Specific Tree Topology, Systematic Biology, 1971, https://doi.org/10.1093/sysbio/20.4.406.
